Bootstrap 5.3 Flexbox Tutorial

What is Flexbox? Flexbox is a one-dimensional layout method for arranging items in rows or columns. Items flex (grow or shrink) to fill additional space or prevent overflow.

1. Basic Flex Container

Use d-flex to create a flexbox container:

<div class="d-flex">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

7. Responsive Flex Classes

Bootstrap flex utilities can be responsive by adding breakpoint prefixes:

<div class="d-flex flex-column flex-md-row">
This div will be a column on small screens and a row on medium screens and up.

9. Quick Reference

Property Bootstrap Class Description
Display Flex d-flex Creates a flexbox container
Flex Direction flex-row, flex-column Sets the direction of flex items
Justify Content justify-content-start/end/center/between/around Aligns items along the main axis
Align Items align-items-start/end/center/baseline/stretch Aligns items along the cross axis
Flex Wrap flex-wrap, flex-nowrap Controls whether items wrap to new lines
Order order-0 through order-5 Changes the visual order of items
Flex Grow flex-grow-0, flex-grow-1 Controls how items grow to fill space
Align Self align-self-start/end/center Overrides alignment for individual items
